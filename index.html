Project 01 · SERVICE DESIGN

당근 지름길

지도앱에는 없는 동네 지름길을, 주민이 직접 발견(Discover)하고 판단(Evaluate)하며 공유(Contribute)하는 UGC 루프로 설계했습니다. 거리별(800m/2km) 플로우 분기와 콜드스타트 시드 전략, 그리고 가설·측정지표·검증방법을 담은 검증 계획까지 관통합니다.

Project 02 · MOBILITY / HMI

Seamless UX

무인 자율주행 택시에서 승객이 느끼는 불안을 낮추기 위해, 차량이 지금 무엇을(Status) 왜(Reason) 어떻게(Action) 하는지 안내하는 프레임을 설계했습니다. 인캐빈 디스플레이와 모바일을 심리스하게 연동하고, 감속·경로변경·긴급정차 세 시나리오로 검증했습니다. Tesla·Waymo·Zoox 티어다운이 기반입니다.

Project 03 · INFORMATION ARCHITECTURE

현대 인증중고차 — 차량 상태 판단 UX 개선

인증중고차 구매자가 흩어진 점검·고지 정보 때문에 차량 상태를 판단하기 어려운 문제를 다뤘습니다. Key First(핵심 먼저) · Visual Clarity(한눈에) · Step by Step(흐름대로) 세 기둥으로 정보구조를 구매 판단 흐름에 맞춰 재설계했습니다.
